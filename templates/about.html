🔬 About FRAUDEX

🎯 Project Overview

FRAUDEX is an advanced Credit Card Fraud Detection System powered by Machine Learning and Artificial Intelligence. It detects fraudulent transactions in real-time with high accuracy to protect financial institutions and cardholders.

🛠️ Technology Stack

  • 🐍 Backend: Python with Flask Web Framework
  • 🎨 Frontend: HTML5, CSS3 with Modern Design
  • 💾 Database: SQLite for Prediction History
  • 🤖 ML Model: Random Forest Classifier with scikit-learn
  • 📊 Data Processing: Pandas & NumPy

📈 Key Features

  • ✅ Real-time fraud detection with 92.4% accuracy
  • ✅ Batch processing of CSV files with multiple transactions
  • ✅ Prediction history and analytics tracking
  • ✅ Confidence scoring for each prediction
  • ✅ User-friendly, modern interface
  • ✅ Secure data handling and processing

🔐 Security & Privacy

All data is processed securely and deleted after analysis. We follow industry best practices for data protection and privacy.
