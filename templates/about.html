Project Description

FRAUDEX is a credit card fraud detection system built for secure, reliable analysis of transaction datasets using a Random Forest machine learning model.

Users register, log in, upload CSVs with transaction features, and review predictions along with analytics over time.

Technology Stack

  • Flask
  • Python
  • SQLite
  • Pandas
  • Scikit-Learn
  • Random Forest
  • HTML/CSS/JavaScript

Project Workflow

CSV Upload → Model Prediction → Fraud Detection → Analytics → Database Storage

Why FRAUDEX

FRAUDEX is designed for internship demonstrations and presentations with a polished cybersecurity theme, secure authentication, and user-specific history.
